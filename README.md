# Other Apps

Lightweight Free and Open Source alternative to Microsoft's Power Apps written in Python and Flask


## Installation

```bash
  git clone https://github.com/OtherExit/OtherApps.git
  cd OtherApps
  python -m venv venv
  source venv/bin/activate
  pip install flask
  python -m flask run
```
